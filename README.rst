=================================================
HybriLIT WORKSHOP 2025 - Miro Ilias working files
=================================================

Working files for quick hand-on-exercises of the HybriLIT WORKSHOP 2025, https://indico.jinr.ru/event/5697.

Users are suposed to have access to Govorun supercomputer to be able to launch prepared SLURM scripts.

Exercises cover free-of-charge computational molecular physics programs MOPAC, ORCA, NWChem, GAMESS-US, DIRAC.

It is also recommended them  to have Avogadro2 (https://two.avogadro.cc/) 
and wxMaxMolPlt (https://brettbode.github.io/wxmacmolplt/MacMolPlt_Manual.html) on their personal computers.
